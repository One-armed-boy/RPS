# RPS
MMR 기반 실시간 웹 가위바위보 매칭 서비스

## Common
- Github Action
- NginX
- Husky

## Server
- TypeScript
- NestJS
- MySQL
- TypeORM
- Redis

## Client
- TypeScript
- ReactJS

## Naming Convention
- class: PascalCase
- function: camelCase
- variable: camelCase
- constants: SNAKE_CASE
- component: PascalCase
- asset, img: SNAKE_CASE
