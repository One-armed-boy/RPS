# RPS
MMR 기반 실시간 웹 가위바위보 매칭 서비스

## Common
- NginX
- TypeScript

## Server
- NestJS
- MySQL
- TypeORM
- Redis

## Client
- ReactJS

## CI/CD
- Github Action
- Husky
- Docker

## Naming Convention
- class: PascalCase
- interface: PascalCase
- function: camelCase
- variable: camelCase
- constants: SNAKE_CASE
- component: PascalCase
- asset, img: SNAKE_CASE
